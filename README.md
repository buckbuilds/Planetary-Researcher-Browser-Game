# Planetary Researcher

A browser-based planetary field research game.

Open `index.html` in a browser to play. The game is self-contained and uses local browser storage for saves.

## Recent Update

- Added anomaly discoveries with evidence-based confirmation.
- Added the Anomalies tab, map markers, Fleet reports, and glossary.
- Improved journal saving for instrument readings.
- Added HUD planet and sky phase information.
